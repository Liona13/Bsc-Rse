# Portfolio Website

A modern portfolio website built with Next.js, Tailwind CSS, and Framer Motion.

## Features
- Responsive design
- Smooth animations
- Glassmorphic UI
- Dark theme
- Interactive elements

## Tech Stack
- Next.js 14
- Tailwind CSS
- Framer Motion
- TypeScript

## Getting Started

First, install the dependencies:

```bash
npm install
```
